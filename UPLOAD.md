# Upload to GitHub Pages

Upload these four files together at the repository root:

- `index.html`
- `sw.js`
- `manifest.webmanifest`
- `icon.svg`

In GitHub, open **Settings → Pages**, choose **Deploy from a branch**, then select the branch and `/ (root)`. Open the published page once while connected so the offline pack can install. Confirm the header reports **Offline pack ready**, then test again in airplane mode before field use.

The editable field profile is stored only in that browser. Reconfirm local stock concentrations after clearing browser data or using a new device.
